# CRM Activity Tracker

A simple web application to track daily activities like calls, meetings, emails, and follow-ups.

## Features
- User Authentication (Signup/Login)
- Add, Edit, Delete Activities
- Activity Status (Open/Closed)
- Search, Filter, Sort
- Dashboard Statistics

## Tech Stack
- HTML
- CSS
- JavaScript
- LocalStorage
